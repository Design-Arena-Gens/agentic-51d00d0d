# A.S Computer-Tech Brand Refresh

A polished single-page experience showcasing an elevated identity system and professional logomark for **A.S Computer-Tech**.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run start
```

## Overview

- Custom SVG logomark with geometric depth and luminous accents
- Responsive layout demonstrating brand context and messaging pillars
- Accessible typography and color system tailored to the refreshed identity

The hero card highlights the new logo while the adjacent copy block articulates positioning and value pillars. The SVG logo is reusable via `public/logo.svg` and the `Logo` component in `app/components/Logo.tsx`.
